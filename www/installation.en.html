Install

How to install foxgame:

  • Method A:
    1. Click on the link.
    2. A yellow message will appear in the top of your firefox telling that it blocked the install. Click on allow.
    3. On the new dialog you will see all domains allowed and a textbox with this domain "http://foxgame.mozdev.org", click on Add.
    4. Click the download link again the install screen will be shown, wait 5 seconds and click on Accept.
    5. Restart firefox to finish the install.
  • Method B:
    1. Left click on the link and select "Save link as..."
    2. Select where the file will be placed.
    3. Exit fullscreen mode (so you can see the file explorer).
    4. Drag and drop the downloaded file inside firefox window.
    5. Restart firefox



How to change foxgame settings:

  • Method A:
    1. In firefox go to "Tools"->"Extensions"
    2. Select foxgame and click on "Settings"
  • Method B:
    1. Left click on any ogame web and select "Configure Foxgame"



How to change firefox languages options:
Warning: Modifying this preference will affect all firefox extensions, it is usefullfor example when you are an italian user that have an english firefox this way you tell firefox that if there is a it-IT locale it should use it.

  • Method A: Install another extension that allows you to switch locales. In http://addons.mozilla.org you can find some of this extensions.
  • Method B:
    1. Type in your firefox address bar "about:config"
    2. Type "locale" in the filter and push filter.
    3. Double click on "general.useragent.locale".
    4. Set it to the desired locale code. Locale codes are usually xx-YY where xx is a language code and YY the country code.
    5. Restart firefox.
